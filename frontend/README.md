# joinchat Frontend
This project was bootstrapped with reactjs, tailwind.

## Available Scripts
In the project directory, you can run:  

### `npm install`
Installing dependency for the app.  

### `npm run start`
Runs the app in the local mode.  

### `npm run build`
The page will build to ready for production.
